1. XSS (Cross-Site Scripting)
Было: В формах и админ-панели данные, введённые пользователем (ФИО, биография, языки и т.д.), выводились напрямую, что позволяло выполнять JavaScript-код.

<p><?=$a['fio']?></p>

Стало: Все пользовательские данные перед выводом обрабатываются функцией htmlspecialchars().

<p><?=htmlspecialchars($a['fio']??'-')?></p>

2. Information Disclosure (Раскрытие информации) 
Было:
	•	Выводились подробные ошибки PHP и PDO (stack trace).
	•	В админ-панели отображались реальные пароли пользователей.
	•	Показывались технические предупреждения.
Стало:
	•	Установлен режим PDO::ERRMODE_SILENT.
	•	При ошибках выводится общее сообщение «Ошибка сервера».
	•	Пароли в админ-панели выводятся, но только для администратора (выделены красным цветом).
$pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_SILENT);

3. SQL Injection
Было: В некоторых местах использовались неподготовленные запросы (хотя в основном Prepared Statements уже применялись).
Стало: Все запросы к базе данных выполняются только через Prepared Statements (prepare + execute). Используются именованные или позиционные параметры. Добавлена функция INSERT IGNORE для предотвращения дублирования.

4. CSRF (Cross-Site Request Forgery)
Было: Отсутствовала защита от поддельных запросов (можно было удалять/изменять данные через вредоносную ссылку).
Стало: Реализована полноценная защита с помощью CSRF-токена:
	•	Генерация токена при старте сессии:
if (empty($_SESSION['csrf_token'])) {
    $_SESSION['csrf_token'] = bin2hex(random_bytes(32));
}
	•	Добавление скрытого поля в каждую форму
  •	Проверка токена перед обработкой POST-запросов (index.php, admin.php, login.php).

5. Include и Upload уязвимости 
Было:
	•	Использовался include('form.php') — потенциальная уязвимость Local File Inclusion.
	•	Отсутствовала загрузка файлов (Upload).
Стало:
	•	include используется только для доверенных локальных файлов.
	•	Путь указывается явно без пользовательских данных.
	•	Загрузка файлов не реализована в приложении, поэтому уязвимость Upload отсутствует.
